Próxima Implementación

Este proyecto será desarrollado utilizando Flask, integrando cálculos estructurales para vigas. El objetivo es crear una herramienta interactiva que facilite la evaluación de cargas y tensiones en vigas de diferentes materiales y dimensiones.

Estamos planificando las funciones principales, entre las que se incluirán:

  • Cálculo de resistencia de materiales.
  • Simulación gráfica de tensiones.
  • Generación de reportes detallados.
